API Endpoints

GET /users

Retrieve all users.

Parameters

  • sort_by (optional) - Field to sort by (username, first_name, last_name)
  • order (optional) - Sort order (asc, desc)

Example Request

GET /users?sort_by=username&order=asc

POST /createUser

Create a new user.

Parameters

  • username (required) - The username of the user
  • password (required) - The password of the user
  • first_name (required) - The first name of the user
  • last_name (required) - The last name of the user

Example Request

POST /createUser
{
    "username": "testuser",
    "password": "Test1234",
    "first_name": "Test",
    "last_name": "User"
}

PUT /users/:id

Update an existing user by ID.

Parameters

  • id (required) - The ID of the user
  • username (required) - The new username of the user
  • password (required) - The new password of the user
  • first_name (required) - The new first name of the user
  • last_name (required) - The new last name of the user

Example Request

PUT /users/1
{
    "username": "updateduser",
    "password": "Updated1234",
    "first_name": "Updated",
    "last_name": "User"
}

DELETE /users/:id

Delete a user by ID.

Parameters

  • id (required) - The ID of the user

Example Request

DELETE /users/1

GET /users/search

Search users by username.

Parameters

  • username (required) - The username to search for

Example Request

GET /users/search?username=testuser

GET /users/searchByDetails

Search users by username, first name, and last name.

Parameters

  • username (optional) - The username to search for
  • first_name (optional) - The first name to search for
  • last_name (optional) - The last name to search for

Example Request

GET /users/searchByDetails?username=testuser&first_name=Test&last_name=User

DELETE /deleteUsersBelowKills

Delete users with less than a certain amount of kills (localhost only).

Parameters

  • kills (required) - The minimum number of kills

Example Request

DELETE /deleteUsersBelowKills?kills=10

GET /statistics

Retrieve all statistics.

Example Request

GET /statistics

POST /addStatistics

Create a new statistic.

Parameters

  • user_id (required) - The ID of the user
  • kills (required) - The number of kills
  • date (required) - The date of the statistic

Example Request

POST /addStatistics
{
    "user_id": 1,
    "kills": 10,
    "date": "2023-01-01"
}

PUT /statistics/:id

Update an existing statistic by ID.

Parameters

  • id (required) - The ID of the statistic
  • user_id (required) - The ID of the user
  • kills (required) - The number of kills
  • date (required) - The date of the statistic

Example Request

PUT /statistics/1
{
    "user_id": 1,
    "kills": 20,
    "date": "2023-01-02"
}

DELETE /statistics/:id

Delete a statistic by ID.

Parameters

  • id (required) - The ID of the statistic

Example Request

DELETE /statistics/1

GET /statistics/paginate

Retrieve statistics with pagination.

Parameters

  • limit (required) - The number of records to retrieve
  • offset (required) - The number of records to skip

Example Request

GET /statistics/paginate?limit=10&offset=0

GET /statistics/search

Search statistics by date range.

Parameters

  • start_date (required) - The start date of the range
  • end_date (optional) - The end date of the range

Example Request

GET /statistics/search?start_date=2023-01-01&end_date=2023-01-31

GET /statistics/searchByDate

Search statistics by a specific date.

Parameters

  • date (required) - The date to search for

Example Request

GET /statistics/searchByDate?date=2023-01-01

GET /testRunning

Check if the server is running.

Example Request

GET /testRunning

Test all endpoints.

Example Request

GET /testEndpoints
